# Delphi e HTML

Integração entre aplicações Delphi e conteúdo HTML/JavaScript.

[![JavaScript](https://img.shields.io/badge/JavaScript-E34F26?style=flat)]()

## Sobre este repositório

Repositório de estudo e prática mantido por [Carlos Baldoino](https://github.com/CarlosBaldoino2005).
Contém exemplos, exercícios ou projetos relacionados a **delphi e html**.

## Tópicos abordados

- Delphi
- HTML
- Web

## Requisitos

| Item | Detalhe |
|------|---------|
| IDE | Embarcadero Delphi (versão compatível com o projeto) |
| SO | Windows (recomendado para projetos VCL/FMX) |
| Dependências | Ver arquivos do projeto (`.dproj`, `boss.json`, etc.) |

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/CarlosBaldoino2005/Delphi-HTML.git
   ```
2. Abra o projeto (`.dproj` ou `.dpr`) no Delphi.
3. Compile e execute conforme as instruções do código-fonte.
